## Type

`object`

## Usage

Keys are identifiers for `fileAttributeOverrides` instances.

```yaml
fileAttributeOverrides:
  main:
    enabled: true
    # ... configuration

  secondary:
    enabled: true
    # ... configuration
```

Available properties:
